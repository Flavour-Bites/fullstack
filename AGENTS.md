# Flavour Bites — Agent Instructions

## Branch Discipline

When committing and pushing changes:

1. **Check if the change belongs to the current branch**
2. **If yes** → commit and push to current branch
3. **If no** → create a new branch with a related name, commit, and push there
4. **Never** make unrelated branches or work on unrelated features
